## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. For the purpose of the hackathon, an API Key has been provided for end users to test the app
4. Run the app:
   `npm run dev`
